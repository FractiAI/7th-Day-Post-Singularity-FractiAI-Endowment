## üéØ Usage Examples

### Example 1: Surgical Operation

```typescript
// Initialize specialist agents
const surgeon = new SurgeonAgent({
  domain: 'surgical',
  expertise: 'cardiac-surgery',
  heroHost: 'leonardo-da-vinci'
});

const assistant = new SurgicalAssistantAgent({
  domain: 'surgical',
  expertise: 'surgical-assistance',
  surgeon: surgeon
});

const recorder = new RecorderAgent({
  domain: 'surgical',
  autoCapture: true,
  generateProtocol: true,
  createSnapshots: true
});

// Begin operation
const operation = await surgeon.beginOperation({
  type: 'cardiac-surgery',
  procedure: 'heart-valve-replacement'
});

// Assistant supports
await assistant.prepareTools(operation);
await assistant.assist(operation);

// Recorder captures everything
await recorder.startRecording(operation);
await recorder.captureObservations(operation);

// Operation completes
await surgeon.completeOperation(operation);

// Recorder generates protocol and snapshot
const protocol = await recorder.generateProtocol(operation);
const snapshot = await recorder.createDiscoverySnapshot(operation);
```
