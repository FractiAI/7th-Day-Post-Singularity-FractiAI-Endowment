## Usage

### Create Next-Octave Protocol

```typescript
import { NextOctaveCreatorStudio } from './src/creator-studio/index.js';

const studio = new NextOctaveCreatorStudio({
  autoBoot: true,
  observerMode: true,
  feedPad: true,
  backwardCompatible: true
});

// Create protocol from observation
const protocol = await studio.createProtocolFromObservation(
  observation,
  {
    targetOctave: AwarenessOctave.TRANSCENDENCE,
    autoLayer: true,
    createSnapshot: true,
    feedToPad: true
  }
);
```

### Layer Protocol

```typescript
// Layer new protocol over existing system
await studio.layerProtocol(protocol, {
  backwardCompatible: true,
  transparent: true,
  safe: true,
  observerMode: true
});
```

// Create snapshot with observer capture
const snapshot = await studio.createSnapshot(protocol, {
